module1.file_a.merge_lists

module1.file_a.merge_lists(*lists)[source]

Merges multiple lists into a single list.

Parameters:

*lists – Variable length argument list of lists to merge.

Returns:

A new list containing all elements from input lists in order.

Return type:

list

Example

>>> merge_lists([1, 2], [3, 4], [5, 6])
[1, 2, 3, 4, 5, 6]
